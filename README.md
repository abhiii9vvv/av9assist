# av9Assist

AI chat assistant built with Next.js (App Router).

## What is included

- Chat UI with message history and local persistence
- Image upload and image generation mode
- Settings, help, analytics, and keyboard shortcuts dialogs

## Quick start

- Install dependencies: `pnpm install`
- Run dev server: `pnpm dev`
- Build: `pnpm build`
- Start production: `pnpm start`

## Project layout

- `app/` App Router pages and API routes
- `components/` shared UI components
- `lib/` providers, utilities, and data helpers
- `public/` static assets

## Configuration

- `next.config.mjs` Next.js config
- `tailwind.config.js` Tailwind config
- `.env*` environment variables (see `.env.local` if present)

## Notes

- Node.js >= 18.17 required
